## üöÄ Getting Started

### Prerequisites
- PHP 7.4 or higher
- Composer for dependency management
- MySQL 5.7 or higher
- Web server (Apache/Nginx)
- SSL certificate for secure payments
- Storage space for digital product files

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/syed-reza98/digital_product_e-commerce.git
   cd digital_product_e-commerce
   ```

2. **Install dependencies**
   ```bash
   composer install
   npm install
   ```

3. **Environment configuration**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Database setup**
   ```bash
   # Configure database and payment gateway credentials in .env
   php artisan migrate
   php artisan db:seed
   ```

5. **Storage configuration**
   ```bash
   php artisan storage:link
   # Set up secure file storage for digital products
   ```

6. **Run the application**
   ```bash
   php artisan serve
   ```
