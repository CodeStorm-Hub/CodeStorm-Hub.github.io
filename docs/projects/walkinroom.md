## üöÄ Getting Started

### Prerequisites
- PHP 8.0 or higher
- Node.js 16+ for JavaScript features
- Composer for PHP dependencies
- MySQL 5.7 or higher
- Web server with SSL support (for payments)
- Email service for notifications

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/syed-reza98/walkinroom.git
   cd walkinroom
   ```

2. **Install dependencies**
   ```bash
   composer install
   npm install
   ```

3. **Environment configuration**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Database setup**
   ```bash
   # Configure database credentials in .env
   php artisan migrate
   php artisan db:seed --class=HotelDataSeeder
   ```

5. **Frontend assets compilation**
   ```bash
   npm run dev
   # For production deployment
   npm run production
   ```

6. **Run the application**
   ```bash
   php artisan serve
   ```
